# IT Manager Command Center

A professional web dashboard for IT managers monitoring external contractors — designed for probation periods, daily reporting, and CEO visibility.

## Live Demo

👉 **[View on GitHub Pages](https://muneerahmad7.github.io/it-manager-dashboard/)**

---

## What's inside

| Tab | What it covers |
|---|---|
| **The 3 People** | Role definitions for NE, SM, and IT Manager |
| **What to Collect** | Daily data checklist from each person |
| **Report Templates** | Copy-ready daily, weekly, and monthly templates |
| **Daily Routine** | Hour-by-hour schedule to own your day |
| **Probation Tips** | 30-60-90 day game plan to convert probation |

---

## How to deploy on GitHub Pages

1. **Fork or upload** this repo to your GitHub account
2. Go to your repo → **Settings** → **Pages**
3. Under **Source**, select `main` branch → `/ (root)` → **Save**
4. Your site will be live at:
   ```
   https://your-username.github.io/it-manager-dashboard
   ```
5. It may take 1–2 minutes to appear the first time

## Features

- Fully responsive — works on mobile and desktop
- 5 interactive tabs covering the complete IT manager workflow
- Copy-to-clipboard on all report templates
- Timeline view for the daily routine
- 30-60-90 day probation game plan
- No dependencies — pure HTML, CSS, JavaScript

---

## Customise it

Edit `index.html` to:
- Add your name to the report templates
- Change the company name in the logo
- Add or remove checklist items for your specific project
